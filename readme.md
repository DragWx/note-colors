# Note Colors
This is a project for experimenting with a system for assigning colors to
musical notes.

It uses YCbCr (related to YUV) to generate a color wheel, and then divides it
into 12 equal hues that are assigned to the 12 semitones in the 12 equal
temperament scale used in most western music.

And then there's a bunch of knobs and settings for playing around with what it
generates and seeing how it looks.
